# معرفو المؤسسات الحكومية

تطبيق هاتف لإدارة هيكلية معرّفي المؤسسات الحكومية في محافظات العراق
(عدا إقليم كردستان: أربيل، السليمانية، دهوك، حلبجة).

## الهيكلية
```
المحافظة (15 محافظة مزروعة مسبقاً)
   └── المؤسسة  →  الاسم + القضاء التابع له
          └── الفرد  →  الاسم + رقم الهاتف + عنوان السكن
```

## المزايا
- إضافة/تعديل/حذف المؤسسات والأفراد.
- اختيار المحافظة ثم المؤسسة عند إضافة فرد (قوائم منسدلة مترابطة).
- **تصفّح هرمي**: المحافظات ← المؤسسات في المحافظة ← المعرّفون في المؤسسة.
- بحث شامل (اسم الفرد، الهاتف، العنوان، اسم المؤسسة، القضاء).
- دعم عربي كامل: تشكيل الحروف لحظياً أثناء الكتابة + اتجاه RTL، شاملاً
  النصوص التلميحية (hint) داخل الحقول.
- أيقونات في الشريط العلوي بدل الأزرار العائمة.
- قاعدة بيانات SQLite محلية، مع حذف متسلسل (حذف مؤسسة يحذف أفرادها).

## التشغيل على الحاسبة (للتجربة)
```bash
pip install -r requirements.txt
python main.py
```
> ضع خطاً عربياً في مجلد `fonts/` أولاً (انظر fonts/README.txt).

## بناء APK للأندرويد

> مهم: نستخدم `pybidi` بدل `python-bidi` في buildozer.spec لأن الأخيرة لا
> تُبنى على أندرويد. الكود يستورد bidi مرناً فيعمل على الحالتين.

### الطريقة الأسهل (بلا إعداد) — GitHub Actions
1. أنشئ مستودعاً على GitHub وارفع إليه كامل المشروع (يتضمن مجلد `.github`).
2. افتح تبويب **Actions** ← شغّل **Build APK** (أو ادفع تعديلاً لفرع main).
3. بعد انتهاء البناء، نزّل ملف الـ APK من قسم **Artifacts** أسفل صفحة التشغيل.

### الطريقة المحلية (لينكس أو WSL2 على ويندوز)
```bash
# 1) أدوات النظام
sudo apt update
sudo apt install -y git zip unzip openjdk-17-jdk python3-pip \
    autoconf libtool pkg-config zlib1g-dev libncurses-dev \
    libncursesw5-dev libtinfo6 cmake libffi-dev libssl-dev build-essential ccache

# 2) أدوات بايثون
pip install --user buildozer cython

# 3) داخل مجلد المشروع (حيث buildozer.spec)
buildozer android debug
```
- البناء الأول يحمّل Android SDK/NDK (عدة جيجابايت) وقد يستغرق 20–40 دقيقة.
- يلزم اتصال إنترنت ومساحة قرص كافية.
- الناتج: `bin/marifin-1.0-debug.apk`.

### تثبيت الـ APK على الهاتف
- انسخ ملف الـ APK إلى الهاتف وثبّته (فعّل «تثبيت من مصادر غير معروفة»)، أو
- عبر USB: `buildozer android debug deploy run` أو `adb install bin/*.apk`.

> على ويندوز مباشرةً لا يعمل buildozer؛ استخدم WSL2 (Ubuntu) أو GitHub Actions.

## ملاحظات تقنية
- **الخط:** التطبيق يسجّل أول ملف خط موجود في `fonts/` باسم "Arabic".
- **العرض العربي:** الدالة `ar()` تستخدم `reshape + bidi`. إن ظهر النص
  معكوساً مرتين على ويندوز فقط، غيّر `USE_BIDI = False` في أعلى `main.py`.
- **الإدخال:** صنف `ArabicInput` يعرض النص الخام أثناء الكتابة (مؤشر سليم)
  والمُشكَّل بعد الخروج، و`get_raw()` يُرجع النص الحقيقي للتخزين.
- **زرع المؤسسات:** يمكن تعبئة `SEED_INSTITUTIONS` في `database.py` لإضافة
  مؤسسات افتراضية عند أول تشغيل.
